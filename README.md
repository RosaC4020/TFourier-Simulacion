# TFourier-Simulacion

Este repositorio contiene simulaciones realizadas en MATLAB Online para analizar señales básicas y su Transformada de Fourier.

## Archivos

- Simulacion_Fourier.m → Script con el análisis completo.
- Gráficas en formato PNG.

## Señales simuladas

- Pulso Rectangular
- Función Escalón
- Señal Senoidal

## Propiedades de la Transformada verificadas

- Linealidad
- Desplazamiento en el tiempo
- Escalamiento en frecuencia
